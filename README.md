# 游戏规则
每手棋双方首先各自选择落子点（允许自尽）并同时亮出。
若双方选择了不同的点，则该手有效，先将这两子落下，结算所有没有气的棋子标记为提子，然后提掉所有提子。
若双方选择了相同的点，则该手无效，该点成为禁入点，双方之后不能再下在该点，直到选择了不同的点。
禁入点可以作为棋子的气。
